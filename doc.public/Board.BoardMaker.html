Module Board.BoardMaker

module BoardMaker: Board 
type opt =
| Miss
| Hit
| Water of Ship.ShipMaker.t option

type opt is the possible types of each space in the board.

exception Overlap
exception Out_of_Bounds
type t

the type implementation of the board, an opt array array.

val make_board : int -> int -> t
val hit : t -> int * int -> unit

hit b pair changes the board b after a coordinate pair is attacked.

val dis_board : t -> unit

dis_board b displays the board in graphical form in the console command window.

val columns : t -> int

columns b gives the number of columns in board b

val rows : t -> int

rows b give the number of rows in board b

val place_ship_h : t -> Ship.ShipMaker.t -> unit

place_ship_h b ship puts ship horizontally into the board b. Raises: Out_of_Bounds if the ship exceeds the bounds of the board. Overlap if the ship will overlap with an existing ship on the board.

val place_ship_v : t -> Ship.ShipMaker.t -> unit

place_ship_h b ship puts ship vertically into the board b. Raises: Out_of_Bounds if the ship exceeds the bounds of the board. Overlap if the ship will overlap with an existing ship on the board.
